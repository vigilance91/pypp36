PyObject* error::set::fromErrno ( PyObject *  type)

convenience function to raise an exception when a C library function has returned an error and set the C variable errno.It constructs a tuple object whose first item is the integer errno value and whose second item is the corresponding error message(gotten from strerror()), and then calls PyErr_SetObject(type, object).

Returns
Alwways NULL, so a wrapper function around a system call can write return PyErr_SetFromErrno(type); when the system call returns an error.
