pxpy

This minimalist C++ library provides a modern, lightweight wrapper for embedding the cPython API into a console application

Developed by [Tyler R. Drury](), with support for the MSVC 2017 compiler, developed with Visual Studios 2017 and Python 3.6.

Use

pxpy is a static link library for use developing C(++) applications executing with an embedded Python interpreter.

along with a copy of python36.dll, the file pxpy36.lib should be included pyConsoleApp.exe someScript.py

then from the top of your project's main file include the line 

#include "pxpy.h";

Then build, compile and run.
