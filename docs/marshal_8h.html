◆ lastObjectFromFile()

PyObject* marshal::read::lastObjectFromFile ( FILE *  file)
Returns
new reference, Python object from the data stream in a FILE* opened for reading.
Note
Unlike PyMarshal_ReadObjectFromFile(), this function assumes that no further objects will be read from the file, allowing it to aggressively load file data into memory so that the de-serialization can operate from data in memory rather than reading a byte at a time from the file.
