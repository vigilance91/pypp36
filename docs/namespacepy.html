int py::atExit ( void(*)()  func)

finalize interpreter, deallocating all resources and disabling any further calls to python functions

Register a cleanup function to be called by Py_Finalize().

Register a cleanup function to be called by Py_Finalize().

  • func will be called with no arguments and should return no value.
    Note
    At most 32 cleanup functions can be registered. The cleanup function registered last is called first. Each cleanup function will be called at most once. Since Python’s internal finalization will have completed before the cleanup function, no Python APIs should be called by func.
    Returns
    when successful returns 0; on failure, returns -1
